去到一个据说宜居的地方 --- 云南

发表于 2016-07-21 | 分类于 走世界

一直都说云南是一个旅游的好地方，很适合居住。所以一直有想去云南旅游的想法，刚好碰上端午节，其他人有假期，就一起来一个说走就走的旅行。马上定下了去云南昆明的火车票。Z211，广州经过广西直达昆明。

行程

广州到昆明

广州火车站虽然看似很大，但是我们去到的时候也有挺多人在准备进站，走入帐篷底下进站，是一个漏斗形进站口，感觉是个坑啊，这种漏斗型的进站口不会最容易导致堵塞么？想想在春运是整个广场都是人的话，该是一个多么辛苦。为了生活，去到一个陌生的城市，为了家人，挨肩擦背在火车站挤火车回家。
火车启动，在晚上醒过了几次，都是在黑夜中，一个城市的黑夜很美，经过广西南宁，外面下着点点的小雨，火车里面的人，大概很多都睡了吧，作为一个互联网从业的一份子，似乎纵有一种很晚睡的习惯，不知是不是子啊创新班中习惯了下来，总要很晚才能睡着。火车站的大叔在为火车补水，火车停了大概20分钟，然后就继续上路，火车在铁轨上走，对于一个第一次搭火车的人来说，一切都是新奇的，行驶中的火车没有想象中嘈杂，深夜行驶的火车还是挺安静的。
早上醒来,已经在云南省内，火车一次又一次的闯过隧道，车窗外的风景一直在变，青山，绿水，一切尽收眼底，以前书上所说的青山，绿水现在有机会可以看到，自己是很是兴奋。一直生活在城镇，也很少见到这种青山绿水。河面在太阳光的照耀下变得金光闪闪。距离终点站越来越近，很多人也醒了。活动在火车中，但相对来说还是挺安静。每个城市都有其不同的特点的，在火车就能看到路途上的东西。
下了火车，出了车站，过马路时没有斑马线的红绿灯。双4车道的马路也没有斑马线的红绿灯，也没有交警协助控制人流。每个城市还是有不同的地方。去了昆明师范大学，学校比较小，但是同样是有林荫的校道，好羡慕。

昆明到大理

第二天就起来就搭了大巴去大理，4个小时的车程，并没有晕车的症状，不知道是气候问题还是我真的变得可以搭车了，并不清楚。不能搭车的人~~，表示搭车是一件很辛苦的事。可是去旅行很多时候都要搭车。经过4个小时车程，来到一个新的城市—大理。下车就被卖猪仔，带到一家客栈，不过价格确实很便宜，环境还挺不错的。放下行李在古城里面闲逛，石板路，中间一条小溪。环境有种说不出的好。还有两只拉布拉多汪星人。6个人在古城里面瞎逛，午餐没有吃，所以晚餐吃得比较早，试了一下烤肉，烤肉做的并不是好吃，烤菜还好。味道都是比较重，都是比较咸的。吃完饭后继续瞎逛，买了鲜花饼，很香的玫瑰香，一个一个都订了要的鲜花饼。在隔离的一家中看到一个驻店歌手收，店里的人都在很享受地听着驻店歌手的歌。有时想，其实生活也可以很简单，一把吉他或许就能走天下了。每个人都会有自己不同的最求，走自己想走的路就对了。大理差不多7点才开始变黑，整个时间活动都推迟了。有更多的白天的时间。晚上回客栈，去了宵夜，喝一杯玫瑰酒，很香玫瑰味，6个人又一起买了这个玫瑰酒。

苍山之行

一个可以乘坐缆车的地方，到达高高的山顶，海拔3900+，山上很凉爽，风景也非常好。爬上又爬下，沿着山顶的路走了很久，很喜欢这个地方，空气又好，又有凉爽的天气，并不像广州那样的闷热。还有一个就是站得高，看得远。由于时间很赶，走完后就直接搭缆车下来。去到山下天龙八部影城还下起了太阳雨，下雨的使得这个景点更加的好看，有一种不知道怎么说的美。为了躲雨走到了一个挂满心愿木板的房子下，每块木板都写上了每个人对自己的生活的期待，看着别人的愿望，觉得都很是美好。希望每个人的生活都能过得很好。回到客栈后，又去古城里面买了一小品玫瑰酒，回到客栈中坐在藤椅上，一手是玫瑰酒，一手是手机，很是舒适，就喜欢的这种静静的坐着，在hea的生活。晚上定了回去的机票。看着机票又577变成677，好可惜啊，快一点就能577啦，便宜100软妹币吖。

双廊

就有在逛逛啊，随便走，围着洱海在汽车

香格里拉

上去有点坑，就是在那边逛逛，然后看了一下赛马场比赛

丽江

好地方，就是在很适合跟妹子一起过来，捂脸跑

去到一个往北的城市

发表于 2016-05-24 | 分类于 走世界

说走就走的旅行

难得的假期，多想出去走走，去自己想去的地方，多去看看其他地方

来到北方城市

来到华中的城市，还好来的时候并不是最热的时候，能让我们有足够的动力爬出去。
day1 广州->武汉
  • 活动范围：中南路附近

9点学校出发，搭高铁去到武汉，历经4个小时，果然还是高铁比较舒适，即使是搭了4小时的高铁也没有很累的感觉，嗯，来到新的地方去走走。12点的高铁，去到武汉就差不多5点了，下车，决定去什么的地方，第一天就想好之后要去什么地方，3个傻瓜对着指引牌对来对去，决定着要去什么地方。

中南路—距离武汉长江大桥（就是那座上走汽车下走火车那个）和黄鹤楼都很近。

武汉的地铁好便宜，搭10+站都是只需要3块。住的地方定下来之后，就考虑去哪里吃东西。主人家说要尝尝小龙虾，一直都听别人说小龙虾很好吃，所以自己也有点心动（yummy yummy），去了一家叫虾蟹馆，不知道是不是早上在车站吃了辣的东西，所以肠胃感觉不是很好。我和另外一个都不是很能吃辣，只有那个小越越比较能吃，不过今天他的胃口也不太好，说是昨晚的酸辣饭惹的祸，（翻白脸！！），所以叫了一个油焖大虾，北方不叫小龙虾，都是叫大虾的。北方很多东西随便吃都是辣的，说的没错，这个油焖大虾其实也是很辣的，好伐！！138一份的小龙虾最后都吃不下了，不过热干膜好像挺好吃，有点习惯这个干干的像大饼什么的东西，不过能沾点什么来吃就更好了。原来的那个调料好咸（毕竟重口味啊）。武昌鱼也很好吃，不过就是骨头有点多，而且骨都好小。吃完出来就肚子不舒服了，像个傻子一个周围找洗手间，最后在一间网吧有洗手间，感谢网吧老板。

day2

  • 活动范围：黄鹤楼，武汉长江大桥，户部巷

    昔人已乘黄鹤去，此地空馀黄鹤楼。
    黄鹤一去不复返，白云千载空悠悠。
    晴川历历汉阳树，芳草萋萋鹦鹉洲。
    日暮乡关何处是，烟波江上使人愁。

    ----《黄鹤楼》崔颢

黄鹤楼经过多次重建，已经变得跟原来的样子不一样了。黄楼楼居然有电梯，当时就懵逼了。经过多次变迁，黄鹤楼被重建了多次。现在的黄鹤楼也就是现代的建筑，有电梯，有灯饰。沿着楼梯向上走，每走一层，都有一种 欲穷千里目，更上一层楼 的感觉。从黄鹤楼能看到不远处的武汉长江大桥，火车呜呜的通过桥体，驶向对岸。站得越高，看的东西也越多，每个人的视觉角度都不同，每个人对同一件事情的看法也有不同，时间不断在过去，需要的是自己好好的活下去。

路上遇到的一只喵喵，一点都不怕人，我们过去撩它，也一直很淡定。闲适的生活吖，喵喵。

next station :have lunch

中午吃的是蔡林记，之前在公司吃的武汉热干面，挺好吃的，现在来到武汉，就一定要试一下。蔡林记的热干面是用芝麻糊来拌面的，味道也不错，加点醋更好吃。

next : 武汉长江大桥

去到江边，有人在横渡，游着游着就飘向下游，也渐渐看不到。这座大桥使用了50+年还是很稳固，外观感觉是欧式的。在附近瞎折腾了1个小时就去户部巷逛了一逛，户部巷是一条食街，小吃什么的好多，食欲不强，所以也没吃什么，而且大部分的小吃都是相似的。

next ：华中科技大学

去到华中科技大学附近找到住的地方，然后在越越的妹妹和他bf的带邻下去吃烤鱼，关谷有好多漂亮的地方，晚饭后就在华中科技大学瞎逛，三个人做电灯泡。学校好大，校道两侧都是好大的梧桐树，夜走华科，

酒店的窗户是在室内的，坑，早上9点窗户外面还是黑漆漆的，还以为是凌晨，-_-||,第三天去逛了白天的华科，两边的梧桐树特别高，树叶把校道都遮盖着，底下全是绿色，这才是校道，好伐，图片迟点附上。

要想更好的思考的话，你需要不断的进行书写

为什么不适用hexo + github 形式

在现在很多的教程中，很多是使用hexo + gitubpage + windows/ubuntu这种形式,但是我并不想实用这种形式的做法，刚好有一个云服务器就拿来试一试，捣鼓一下

使用hexo缘由

原本是使用wordpress进行博客的编写，但是使用wordpress时，有很多东西都可以自己设置，但是wordpress每次在打开是总会有各种说要更新什么的

这篇文章说的是什么

将hexo搭建在ubuntu系统上，使用通过搭建git来完成自动部署，并且使用nginx来搭建服务器

怎么做

  1. 安装nodejs
    由于使用hexo使用node来进行静态页面的生成，所以需要安装nodejs
    通过apt-get进行安装

    `sudo apt-get install nodejs`
`sudo apt-get install npm`

  2. 安装git
    通过git的post-receivehook，来进行自动部署
    sudo apt-get install git

  1. 安装hexo

    npm install hexo-cli -g

ssh在window下向ssh服务器传递文件

发表于 2016-05-17 | 分类于 ubuntu

上传文件

在window平台上传文件到指定的ubuntu服务器

  1. 首先，你需要有一个可以执行linux命令的工具
  2. 在执行工具中执行scp命令

在window下，我是使用git Bash来进行操作的

使用的是scp命令

scp file username@host:./path/filename

以上的命令是将本地的文件上传到host所指定的服务器中，具体的放置目录是user的主目录的path目录，并将文件filename进行命名

上传文件夹

scp -r location username@host:./path/flodername

在该命令下增加了-r选项是表示以递归的形式上传文件夹，就是上传整个文件夹将上传的文件夹以filename进行命名

下载文件

scp username@hostname:path/filename1 /filename2

将服务器中username用户目录下的filename1文件复制到本地的目录，并且以filename2命名

下载文件夹

scp -r username@hostname:/flodername /path

将远程服务器上的文件下载到本地的 /path目录下

在ssh服务器之间传输文件

sftp
1. sftp与ftp传输文件不同
2. sftp是使用ssh来进行文件的上传以及下载，使用shh作为传输底层要比ftp传输安全

首先连接远程服务器

sftp username@host

连接服务器，默认使用端口22

  1. 下载文件

    get file

  2. 上传文件

    put file
