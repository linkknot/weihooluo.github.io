Ubuntu安装ffmpeg

由于视频处理需要，需要在ubuntu上部署ffmpeg，整体来说流程有以下方法：

  1. 直接通过apt-get安装，但是不推荐，安装的版本的会很旧
  1. 自己编译，这也是重点说的。

自己编译ffmpeg

  1. 下载

    找目录作为下载目录

  2. 安装需要的库

  3. 编译

    由于不同的版本的要求，可能会出现报./configure中没有某个选项，这时候需要删除某些库，下面是删除libfaac，x11grab，所以可以使用下面的命令

    make
